Additional Connection Settings

This is last page of connection wizard and it contains properties common for all drivers.

These properties are used by DBeaver itself, not by drivers.

Connection Name Name of connection. It will be used in database navigator.
Save password locally Check this field if you want to store database password locally. Otherwise DBeaver will ask password each time you connect to this database. Passwords are stored in encrypted form, but if you use highly secured databases and it is possible that third person can access your computer we recommend to avoid this feature.
Show system objects Check if you want to see database system objects (schemas, tables, etc). If not checked then system objects will be hidden.
Filter catalogs/schemas Show only catalogs (schemas) which name matches this pattern. Pattern format may be specific to each database but generally it has LIKE syntax.
