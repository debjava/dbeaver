Connection Settings

Basic settings

Here user may specify connection settings and properties.

Some properties are required, other are optional.

List of standard settings:

JDBC URL Database URL. This field is editable only if database driver doesn't have sample URL. Otherwise it is not editable and it's value will be automatically change whenever you change other connection settings.
Host Database host name
Port Database port number. Usually this field is optional. If left blank then default database port will be used.
Database/Schema Name of remote database/schema.
Server Name of remote server.
User name/password User credentials. May be left blank for some drivers (e.g. for embedded databases)

Advanced settings

Database driver may support set of additional connection settings. User mey view list of all available properties on "Advanced" tab. All properties are editable.

Some drivers supports additional properties but doesn't provide list of them. In this case you may find list of available properties in driver's documentation and then specify any number of properties under "User Properties" element. Right click on it and choose "Add new property" menu item.
