PostgreSQL
	- getTables() возвращает не только таблицы но и индексы с сиквенсами (возможно что-то ещё)
	- после ошибки (SQLException) надо выполнить rollback чтобы делать новые селекты

ODBC
	- нельзя дважды фетчить одну и ту же колонку (ошибка no data found)
	- getCatalogs возвращает список с другими именами

MySQL
	- резалтсет всегда приходит целиком с сервера. большие таблицы надо обрезать по setMaxRows
