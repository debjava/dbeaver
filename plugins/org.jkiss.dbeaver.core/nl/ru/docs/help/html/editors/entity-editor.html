Редактор сущности

Позволяет Вам просматривать и редактировать объекты базы данных.

Редактор сущности состоит из нескольких закладок. Различные объекты имеют разные наборы закладок. Кроме того, дополнительные закладки могут предоставляться плагинами. Одна стандартная закладка, которая представлена для большинства типов объектов - это закладка "Свойства".

Свойства объекта

Также содержит несколько закладок. Первая и единственная стандартная закладка - это непосредственно набор свойств объекта. Другие закладки зависят от типа объекта и описывают его структуру.
