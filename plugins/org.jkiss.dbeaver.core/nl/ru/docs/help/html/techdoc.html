Техническая информация

DBeaver основан на платформе Eclipse (3.7). Он частично использует исходные коды следующих компонент:

  • Платформа Eclipse
  • javahexeditor - двоичный вьюер
  • Nebula grid - таблица выборки, была практически полностью переписана
  • Silk набор иконок

Windows-инсталлятор создан с помощью NSIS

Debian-пакет создан с помощью ant-deb-task

DBeaver - это часть проекта JKISS.
