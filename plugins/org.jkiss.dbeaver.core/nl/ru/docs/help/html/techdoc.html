Technical Information

DBeaver is based on Eclipse(3.6) platform. It partially uses sources of the following components:

  • Eclipse framework
  • javahexeditor - binary viewer
  • Nebula grid - result set grid, was totally rewritten
  • Silk icon set

Windows installer created with NSIS

Debian package created with ant-deb-task

DBeaver is a part of JKISS project.
