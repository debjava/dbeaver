DBeaver Roadmap 2011

This is very brief version of readmap 2011. Just primary goals in order of priority.

Metadata Editor

  1. Common table editor. Visual editor of relational tables. Columns, indexes, constraints, options.
  2. Implementation of metadata editor in Generic, MySQL, Oracle, PostgreSQL plugins.

Database plugins

Oracle

  1. Support of standard meta objects (tablespaces, triggers, packages, etc).
  2. Support of oracle specific datatypes (XML, etc).

PostgreSQL

  1. Support of standard meta objects (tablespaces, sequences, etc).
  2. Support of PostgreSQL specific datatypes (arrays, etc).

Non-relational

  1. WMI (WBEM?) as first non JDBC datasource.
  2. Try to adapt hierarchical/multivalue databases like jBase and ADABAS. Need access to working instances.

Help

  1. Integrated help system
  2. Context help for dialogs/editors

I18N

  1. Add i18n support in all dialogs/editors/exceptions
  2. Localize help

Views

  1. Edit/compile views.

Stored Procedures

  1. Supports of edit, compile and debug of any stored code (procedures, triggers).
