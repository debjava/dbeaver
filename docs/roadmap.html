DBeaver Roadmap 2011

Oracle plugin

  1. Oracle: Support of standard meta objects (tablespaces, triggers, packages, etc).
  2. Oracle: Support of oracle specific datatypes (XML, etc).

Tables Editor

  1. CORE: Common table editor. Visual editor of relational tables. Columns, indexes, constraints, options.
  2. GENERIC: Generic table editor implementation. Table editor which supports only SQL92 constructs.
  3. MySQL: MySQL table editor implementation.

Stored Procedures

  1. CORE: stored code (procedures, functions) editor. Supports edit, compile, debug.
  2. MySQL: stored procedures editor implementation.
