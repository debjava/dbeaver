DBeaver Roadmap 2011

Oracle plugin

  1. Oracle: Support of standard meta objects (tablespaces, triggers, packages, etc).
  2. Oracle: Support of oracle specific datatypes (XML, etc).

PostgreSQL plugin

  1. Postgres: Support of standard meta objects (tablespaces, sequences, etc).
  2. Postgres: Support of PostgreSQK specific datatypes (arrays, etc).

Tables Editor

  1. CORE: Common table editor. Visual editor of relational tables. Columns, indexes, constraints, options.
  2. GENERIC: Generic table editor implementation. Table editor which supports only SQL92 constructs.
  3. MySQL: MySQL table editor implementation.
  4. Oracle: Oracle table editor implementation.

Stored Procedures

  1. CORE: stored code (procedures, functions) editor. Supports edit, compile, debug.
  2. MySQL: stored procedures/triggers editor implementation.
  3. Oracle: stored procedures/triggers/packages editor implementation.

ERD

  1. ERD: emprove ERD layouting.
  2. ERD: allow custom ERD edit/load/save.

I18N

  1. I18N: Add i18n support in all dialogs/editors/exceptions
  2. I18N: Localize help
