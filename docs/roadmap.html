DBeaver Roadmap 2011

New plugins

Oracle

  1. Support of standard meta objects (tablespaces, triggers, packages, etc).
  2. Support of oracle specific datatypes (XML, etc).

PostgreSQL

  1. Support of standard meta objects (tablespaces, sequences, etc).
  2. Support of PostgreSQK specific datatypes (arrays, etc).

WMI

  1. Implement first non JDBC datasource.
  2. WMI metadata hierarchy.
  3. WMI queries (readonly).

I18N

  1. I18N: Add i18n support in all dialogs/editors/exceptions
  2. I18N: Localize help

Tables Editor

  1. Common table editor. Visual editor of relational tables. Columns, indexes, constraints, options.
  2. Generic: Generic table editor implementation. Table editor which supports only SQL92 constructs.
  3. MySQL: MySQL table editor implementation.
  4. Oracle: Oracle table editor implementation.

Stored Procedures

  1. Supports of edit, compile and debug of any stored code.
