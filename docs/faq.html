Frequently Asked Questions about DBeaver

General

Why "DBeaver"?

DBeaver named in contrast to SQuirreL, popular java-based database client. It's like "database beaver against SQL squirrel". Also I just like these animals :)

Who makes it and for what reason?

Check Team page to see who we are.

For now there is no any certain goal of this project. We just want to make a good (better than ony other) universal tool to work with databases.

This is a completely non-commercial project, nobody pays us for the job, there are no donations or any other kind of funding. This project is powered by our pleasure for 100%.

So far.

May I access DBeaver source code?

Not yet. At the moment DBeaver is completely freeware but it is not opensource.

Installing and running

DBeaver doesn't start

A1: You have old Java or not Java VM installed. DBeaver require Java 1.6. If you have different versions of JVM on your computer then you may specify particular one in command line (see readme file about command line arguments).

A2: You use wrong DBeaver version (e.g. 64bit on 32bit OS)
